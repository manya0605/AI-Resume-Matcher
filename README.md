# 🤖 AI Resume Matcher

An AI-powered web application that analyzes how well a **resume matches a job description** using **Natural Language Processing (NLP)** techniques.

The system calculates a **match score** and highlights **matching keywords**, helping job seekers optimize their resumes for specific job roles.

---

## 🚀 Demo

This application allows users to:

* Paste their **resume**
* Paste a **job description**
* Calculate a **resume-job similarity score**
* View **matching keywords**

---

## 🧠 How It Works

The project uses **Natural Language Processing (NLP)** and **Machine Learning techniques**:

1. Convert resume and job description text into numerical vectors using **TF-IDF Vectorization**
2. Compare both texts using **Cosine Similarity**
3. Generate a **match percentage**
4. Extract **common keywords** between both texts

## ⚙️ Installation

Clone the repository

git clone https://github.com/your-username/AI-Resume-Matcher.git

Move to the project folder

cd AI-Resume-Matcher

Install required libraries

pip install -r requirements.txt

---

## ▶️ Run the Application

Start the Streamlit app

streamlit run Job_resume_matcher.py

The app will automatically open in your browser.
